# Music Review App

Fullstack application to review songs, built with:

- Backend: Golang (Gin, GORM)
- Frontend: React
- Database: PostgreSQL
- Deployment: Docker Compose

## Structure

- `/backend` - Go API using Gin
- `/frontend` - React SPA
- `/db` - Database-related files (e.g., migrations)
