## Built with

- Backend: Golang (Gin, GORM)
- Frontend: React
- Database: PostgreSQL

## Structure

```
/arbust-app
 ├── /backend     # Go API using Gin
 ├── /frontend    # React SPA
 └── /db          # DB related files (e.g., migrations)
```

## Setup

1. Copy `.env.example` to `.env`.
2. Fill up with desired credentials.
3. Run `docker compose up -d`.
